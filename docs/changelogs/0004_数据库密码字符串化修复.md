# 变更日志: 数据库密码字符串化修复
**日期**: 2025-12-09
**任务**: 确保 TypeORM 传递给 PostgreSQL 的密码为字符串并提供默认配置

## 📂 文件变更
### 修改 (Modified) / 新增 (New)
- `[src/config/configuration.ts]`:
  - 数据库配置增加默认值，端口仅 parseInt，密码保留字符串且提供默认值；JWT 配置增加默认密钥。
- `[src/app.module.ts]`:
  - TypeORM 异步工厂显式类型化 `TypeOrmModuleOptions`，为数据库配置设定默认值，并使用 `String()` 强制密码为字符串，防止传入 undefined/number。

## 💡 技术说明 (Technical Notes)
- **关键点**: 强制数据库密码字符串化，避免 `SASL: client password must be a string` 错误；为数据库/JWT 配置添加安全默认值。
- **配置变更**: `.env` 未改动，但需确保包含 DB_HOST/DB_PORT/DB_USERNAME/DB_PASSWORD/DB_DATABASE。
